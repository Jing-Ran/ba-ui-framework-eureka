Inline Note Taking

To create a new note, select some text and use keyboard shortcut Shift+Control+I. Type some notes into the note area.

To close a note, click the selected text or the window other than the note area. To reopen a note, click the selected text.

To delete a note, open the note, then click the "remove" button or use keyboard shortcut Shift+Control+D. Click "Clear All Notes" to remove all.

To print all notes, click "Print Note" button.

NOTE: for now you can only add note to text that is inside of a single HTML tag.
